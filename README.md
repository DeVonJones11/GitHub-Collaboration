# Java Calculator Project

## Description
This is a simple console-based calculator program written in Java. It allows users to perform basic arithmetic operations such as addition, subtraction, multiplication, and division by entering numbers and selecting an operation.

## Features
- Addition of two numbers  
- Subtraction of two numbers  
- Multiplication of two numbers  
- Division of two numbers (with divide-by-zero protection)  
- User input using Scanner  
- Menu-based selection system  

## Technologies Used
- Java  
- Scanner class  
- Git & GitHub
